# ✨ Era Uma Vez

Site profissional desenvolvido para a marca **Era Uma Vez**, com foco em presença digital, divulgação dos serviços e fortalecimento da identidade visual do negócio.

---

## 📌 Sobre o Projeto

O projeto **Era Uma Vez** foi criado para atender um cliente real, oferecendo uma solução moderna, elegante e responsiva para apresentar a empresa na internet de forma profissional.

O objetivo principal foi transformar a marca em uma presença digital forte, transmitindo confiança e facilitando o contato com novos clientes.

---
